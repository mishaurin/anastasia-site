# Anastasia Mishurina — Сайт визажиста

Официальный сайт профессионального визажиста Анастасии Мишуриной.

## Структура файлов

```
anastasia-site/
├── index.html          ← Главная страница сайта
├── photos.json         ← Список фотографий портфолио
├── netlify.toml        ← Настройки хостинга
├── README.md           ← Этот файл
└── photos/
    ├── hero/           ← Фото для главной карусели
    │   ├── hero-1.jpg
    │   ├── hero-2.jpg
    │   ├── hero-3.jpg
    │   ├── hero-4.jpg
    │   └── hero-5.jpg
    ├── mariage/        ← Свадебные фото
    ├── gala/           ← Гала и вечерние мероприятия
    ├── tv/             ← Телевидение и интервью
    └── shooting/       ← Фотосъёмки
```

## Как добавить новое фото в портфолио

1. Загрузите файл в нужную папку (`photos/mariage/`, `photos/gala/` и т.д.)
2. Откройте файл `photos.json` и добавьте новую строку:

```json
{
  "file": "имя-файла.jpg",
  "category": "mariage",
  "caption_fr": "Подпись на французском",
  "caption_en": "Caption in English"
}
```

3. Нажмите **Commit changes** — сайт обновится автоматически за 30–60 секунд.

## Как удалить фото из портфолио

1. Откройте `photos.json`
2. Удалите строку с нужным фото
3. Нажмите **Commit changes**

Сам файл из папки `photos/` можно тоже удалить — но это необязательно, на отображение сайта не влияет.

## Как обновить фото в карусели на главной

Просто замените файлы `photos/hero/hero-1.jpg` ... `hero-5.jpg` на новые с теми же именами.

## Категории портфолио

| Категория в JSON | Папка | Отображается как |
|-----------------|-------|-----------------|
| `mariage` | `photos/mariage/` | Mariage |
| `gala` | `photos/gala/` | Gala & Soirée |
| `tv` | `photos/tv/` | TV & Interview |
| `shooting` | `photos/shooting/` | Shooting |

## Рекомендации по фотографиям

- **Формат:** JPG
- **Минимальный размер:** 1200px по ширине
- **Вес файла:** до 500 Кб (сжимайте через [Squoosh](https://squoosh.app) — бесплатно)
- **Имена файлов:** только латиница, без пробелов и спецсимволов (используйте дефис `-`)

**Примеры правильных имён:** `chateau-bagnols-2024.jpg`, `cannes-gala.jpg`  
**Примеры неправильных:** `Свадьба Château.jpg`, `фото 1.jpg`
